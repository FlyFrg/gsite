Нейминг

[local/blockedout/blockedin]_[geoip/geosite]_[country].dat

Типы
* local - доступны локально из страны [country] например google.com ДОСТУПЕН напрямую(без прокси) из TM
* blockedout - заблокированы на ВЫХОД из страны [country], например linkedin.com ЗАБЛОКИРОВАН из RU, например список блокировок РКН
* blockedin - заблокированные на ВХОД в стране [country], например mysite.cn медленен/ЗАБЛОКИРОВАН из не CN, так же все китайские сайты
